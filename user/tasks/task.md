# Task Management Template

> A simple markdown template for tracking work and project tasks

---

## Quick Reference

**Task Status:**
- `- [ ]` Incomplete task
- `- [x]` Completed task
- `- [~]` In progress (optional)
- `- [-]` Blocked/cancelled (optional)

**Common Tags:**
- `@high` `@medium` `@low` - Priority levels
- `@urgent` - Time-sensitive tasks
- `@blocked` - Waiting on dependencies
- `@review` - Needs review or approval
- `@bug` `@feature` `@docs` - Work type
- `@team-name` - Team or person assignment

---

## Active Tasks

### Current Sprint / This Week

- [ ] Task name here @high @feature
  - **Notes:** Additional context, links, or details about this task
  - **Due:** YYYY-MM-DD

- [ ] Another task example @medium
  - **Notes:** Any relevant information or blockers

- [ ] Simple task without notes @low


### Backlog

- [ ] Future task to prioritize later
- [ ] Another backlog item @feature

## Completed Tasks

- [x] Example completed task @high
  - **Completed:** 2026-01-15
  - **Notes:** Brief summary of outcome or results

## Blocked / On Hold

## Tips for Using This Template

1. **Copy sections** as needed for different projects or time periods
2. **Archive completed tasks** regularly to keep the file manageable
3. **Use tags consistently** to make searching easier
4. **Add links** to tickets, docs, or related resources in the notes
5. **Date your tasks** when time-sensitivity matters
6. **Review weekly** to update priorities and clear completed items
